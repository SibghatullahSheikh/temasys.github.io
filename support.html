  • Trouble getting things to work?

    Ah, bummer! Here are a few things you can do:

    1. Our projects usually include tests. Please make sure they run successfully on your system. You'll find instructions on how to run them in the projects description or README.md file.
    2. Checkout the Issues section of the project you're using and see if there's already an issue on file.
    3. Our staff hangs out a lot on stackoverflow.com. If you create a question there with the project name as tag (e.g. skywayjs or adapterjs) we'll find it and reply within a working day. This way you're also helping the community, if they're having a similar problem in the future.
    4. Worst case, write us an email directly to dev-support at temasys.com.sg
